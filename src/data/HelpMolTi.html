MolTi


MolTi allows identifying communities from a collection of input networks (i.e. a multiplex network). MolTi is based on the Louvain algorithm (Blondel et al. 2008) adapted to optimize the Multiplex-modularity (Didier et al. 2015).


Load Graphs panel

Load different networks as input from your computer. Network format are simple double-column,"ncol" format (one interaction per line, the 2 interactors being separated by a tab or a space). The node names need to in the same format in the different network.








Parameters panel

Set the resolution parameter, which allow reducing the sizes of the obtained communities, when the input networks are large.


(Optional) Community annotation panel

MolTi can compute annotation enrichments for each community of the obtained partition. Enrichments are computed with the Fisher exact test, and multiple testing is corrected with the Bonferroni approach.

- Set here the q-value threshold for an enrichment to be kept as significant.

Notice that the background values for the Fisher exact test are computed on this annotation file.

- A description file, containing the full description of the annotations can be provided to get a more comprehensive version of the term annotations.







Clicking on the toolbar button launches the community detection. The communities are displayed in the left table in which you can navigate and obtain additional information. The user can enter a gene/protein name in the text field below the community panel to select the community containing this gene/protein. In particular, if you have loaded an annotation file, the terms enrichment of each community are computed. The most significant Fisher statistic among terms is displayed for all the communities and more complete information about the community selected is provided in the bottom text panel.
